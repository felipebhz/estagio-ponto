Especificações:

Linguagem: PHP
Framework: Code Igniter 3 - Last Stable
Banco de Dados: MySQL 5.7

1 - Para configurar este sistema em seu ambiente, você deverá alterar alguns arquivos:

* /application/config/config.php configure a BASE_URL para o seu hostname.
* /application/config/database.php configure os dados de conexão ao banco como host, usuário e senha.
* Há um dump do database na raiz do sistema para que possa montar o banco em seu ambiente.

Em alguns servidores com Apache e CentOS, você poderá ter que colocar o index.php em sua BASE_URL para que os links da aplicação funcionem adequadamente, pois restrições de cada sistema Linux + Apache podem ser diferentes.
